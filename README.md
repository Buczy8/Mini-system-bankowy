### Komendy Git użyte do zachowania projektu:
1. Inicjalizacja repozytorium:
   `git init`
2. Dodanie wszystkich plików bez tych z .gitignore do śledzenia:
   `git add .`
3. Zapisanie zmian wiadomością:
   `git commit -m "Użyte komendy git"`
4. Wypchnięcie zapisanych zmian do zdalnego repozytorium:
    `git push -u origin master`
